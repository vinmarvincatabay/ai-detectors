# Filipino AI & Similarity Detector

## Deskripsyon
Ito ay isang web app na sumusuri ng teksto kung may AI-generated content at nagche-check ng similarity ng dalawang texts sa Filipino. Ginawa gamit ang **Streamlit**.

## Features
- AI-likelihood scoring (approximate, rule-based)
- Similarity checker gamit ang Jaccard similarity
- File upload: `.txt`, `.docx`, `.pdf`
- Simple login system (demo account: `demo/demo123`)
